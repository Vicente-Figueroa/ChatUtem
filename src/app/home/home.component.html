Proyecto Chatbot UTEM

Bienvenido al proyecto de chatbot desarrollado para la Universidad Tecnológica Metropolitana (UTEM).

Tecnologías utilizadas

  • Angular
  • Flask
  • Heroku
  • Modelo Gemini 1.5 Flash

Proceso de desarrollo

  • Web scraping a la página de la UTEM
  • Vectorización de datos en una base de datos embebida
  • Almacenamiento de archivos en AWS
  • Optimización para respuestas rápidas del servidor

Este proyecto fue desarrollado como parte de una iniciativa para mejorar la interacción y el acceso a la información en la UTEM.
